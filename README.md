# 🔐 Password Generator

A modern and responsive **Password Generator Web App** built using **HTML, CSS, and JavaScript**.  
This project helps users generate secure passwords with customizable options like uppercase letters, lowercase letters, numbers, and symbols.

---

## ✨ Features

✅ Generate strong random passwords  
✅ Adjustable password length slider  
✅ Include/exclude:
- Uppercase letters
- Lowercase letters
- Numbers
- Symbols

✅ Copy password to clipboard  
✅ Password strength indicator  
✅ Responsive UI  
✅ Clean glassmorphism design  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## ⚙️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/vaibhav-070/password-generator.git
```

2. Open project folder

```bash
cd password-generator
```

3. Run the project

Simply open `index.html` in your browser.

---

## 🚀 How It Works

- User selects password length  
- Chooses desired character types  
- Clicks **Generate Password**  
- App creates a random secure password  
- User can copy it instantly
